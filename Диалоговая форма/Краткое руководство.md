# Как пользоваться системой: краткое руководство

## Общая логика

Ты проходишь 4 шага. Каждый шаг — в **отдельном чате** с ИИ.
На выходе получаешь файл с результатами. Эти файлы переиспользуешь на следующих шагах.

**Важно:** не пытайся пройти всё в одном чате. Отдельный чат на шаг = чистый контекст и точный результат.

---

## Шаг 1. Мой профиль

**Где проходить:** в новом чате.
**Что делаешь:** копируешь инструкцию для Шага 1 и отправляешь ИИ.
**Что происходит:** ИИ задаёт вопросы про твои триггеры и реакции. Ты отвечаешь.
**Что получаешь на выходе:** markdown-файл с твоим конфликтным профилем.

**Инструкцию для Шага 1 можно запросить у меня.**

---

## Шаг 2. Профиль коллеги

**Где проходить:** в новом чате.
**Что делаешь:** копируешь инструкцию для Шага 2 и отправляешь ИИ.
**Что происходит:** ИИ опрашивает тебя про одного коллегу.
**Что получаешь на выходе:** markdown-файл с профилем этого коллеги.

Если коллег несколько — проходи Шаг 2 для каждого в отдельном чате или повтори цикл в том же чате (удобно держать отдельно).

**Инструкцию для Шага 2 можно запросить у меня.**

---

## Шаг 3. Структура компании

**Где проходить:** в новом чате.
**Что делаешь:** копируешь инструкцию для Шага 3 и отправляешь ИИ.
**Что происходит:** ИИ опрашивает тебя про иерархию, зоны власти и системные конфликты.
**Что получаешь на выходе:** markdown-файл с картой компании.

**Инструкцию для Шага 3 можно запросить у меня.**

---

## Шаг 4. Рисковая ситуация

**Где проходить:** в новом чате.
**Что делаешь:** берёшь файлы из шагов 1–3 и отправляешь их в новый чат вместе с инструкцией для Шага 4.
**Что происходит:** ИИ читает твои профили, а потом в диалоге опрашивает тебя про конкретную ситуацию.
**Что получаешь на выходе:** 2–3 варианта действий с конкретными фразами, прогнозом реакции оппонента и рисками.

**Инструкцию для Шага 4 можно запросить у меня.**

---

## Как начать чат 4 (самый важный момент)

В новый чат ты отправляешь **три файла** (или просто копируешь их содержимое):
- Шаг 1. Мой профиль
- Шаг 2. Профиль коллеги (или несколько)
- Шаг 3. Структура компании

И следом — **инструкцию для Шага 4**, которая говорит ИИ, что делать с этими данными.

Без файлов из первых трёх шагов четвёртый работать не будет — ИИ просто не будет знать твои триггеры, мотивы коллеги и контекст компании.

---

## Коротко: что тебе понадобится

1. Ссылки или тексты инструкций для шагов 1–4 (можно запросить у меня).
2. Отдельные чаты для шагов 1, 2, 3, 4.
3. Файлы на выходе.
4. Один финальный чат, куда ты кладёшь все файлы + инструкцию для Шага 4.

Всё.
